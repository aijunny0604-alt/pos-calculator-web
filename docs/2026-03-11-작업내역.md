# 2026-03-11 작업 내역

## 1. 주문 편집 저장 버그 수정

### 문제
- 주문 상세에서 제품 삭제 후 저장해도 반영되지 않음

### 원인
- `OrderDetail.jsx`의 `handleSave`가 `onUpdateOrder(updatedOrder)`로 전체 객체를 1개 인자로 전달
- `App.jsx`의 `handleUpdateOrder`는 `(id, data)` 2개 인자를 기대

### 수정
- **OrderDetail.jsx**: `onUpdateOrder(order.id, { items, customer_name, customer_phone, customer_address, total, subtotal, vat, memo })` 형태로 변경
- **App.jsx**: `handleUpdateOrder`에서 snake_case → camelCase 동기화 추가 (`data.total` → `merged.totalAmount` 등)
- **App.jsx**: `selectedOrder` 상태도 함께 업데이트하도록 수정

## 2. 장바구니 → 주문 전환 시 0원 버그 수정

### 문제
- 저장된 장바구니에서 "주문 등록"하면 금액이 0원으로 저장됨

### 원인
- 저장된 장바구니 아이템은 `wholesale`/`retail` 필드 사용
- `onOrder` 핸들러에서 `i.price || 0`으로 계산 → `price` 필드가 없으므로 항상 0

### 수정
- **App.jsx** `onOrder` 핸들러: `price_type`에 따라 `wholesale` 또는 `retail` 필드로 폴백
```javascript
const price = i.price || (pt === 'wholesale' ? (i.wholesale || 0) : (i.retail || i.wholesale || 0));
```

## 3. 번웨이 차량 모델 관리 시스템 통합

### 문제
- 관리자 페이지에서 추가한 커스텀 모델이 번웨이 메인 페이지에 안 나타남
- 기본 모델(벨로스터N, 아반떼N, 스팅어)은 수정/삭제 불가

### 변경 내용

#### AdminPage.jsx
- **저장 방식 통합**: 기본 모델 + 커스텀 모델을 `localStorage('burnway-car-models')` 단일 키로 관리
- **`loadBurnwayModels()`**: 통합 로드 + 기존 `burnway-custom-car-models` 키에서 자동 마이그레이션
- **`detectBurnwayCarModel()`**: 키워드 기반 매칭으로 변경 (하드코딩 제거)
- 모든 모델에 **수정**(Edit 아이콘) / **삭제**(Trash2 아이콘) 버튼 추가
- 수정 시 인라인 편집: 모델명 + 키워드(쉼표 구분) 입력 가능

#### BurnwayStock.jsx
- 하드코딩된 `CAR_MODELS` 배열 제거
- `localStorage('burnway-car-models')`에서 모델 로드
- `detectCarModel()` → 키워드 기반 매칭으로 변경
- 페이지 진입 시 + focus 이벤트 시 모델 목록 새로고침 (관리자→번웨이 이동 시 즉시 반영)

## 4. 자바라 세트 속성 누락 버그 수정

### 문제
- 모델 통합 후 자바라 관련 UI(세트 현황, 자바라 재고 등)가 사라짐

### 원인
- `AdminPage.jsx`의 `BURNWAY_INITIAL_MODELS`에 `hasJabara`/`hasDctManual` 속성이 빠져있었음
- localStorage에 속성 없이 저장 → BurnwayStock에서 로드 시 자바라 분류 불가

### 수정
- 양쪽 파일의 `BURNWAY_INITIAL_MODELS`에 `hasJabara`/`hasDctManual` 추가
- `loadBurnwayModels()`에 복원 로직 추가: 기존 localStorage 데이터에 속성이 없으면 기본값에서 자동 복원

## 5. 자바라 세트 옵션 관리자 UI 추가

### 기능
- 모델 **추가** 시: 모델명 입력란 아래에 "자바라 세트" 체크박스
  - 체크하면 "DCT/수동 구분" 체크박스 추가 노출
- 모델 **수정** 시: 동일한 체크박스 2개 표시, 기존 모델도 세트 on/off 변경 가능
- 모델 목록: 자바라 세트 활성화된 모델에 "세트" 뱃지 표시

### 동작
- "자바라 세트" 체크 → 해당 차종의 번웨이 카드에서 `완성 세트` 계산 활성화
- "DCT/수동 구분" 체크 → 벨로스터N처럼 DCT 자바라 / 수동 자바라를 따로 분류
- 체크 해제 시 → 단순 촉매/직관 재고만 표시 (세트 계산 없음)
